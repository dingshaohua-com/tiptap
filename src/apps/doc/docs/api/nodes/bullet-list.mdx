该扩展会使编辑器识别`<ul>`标签。
在新行的开头输入`*`，`-`，`+`会将该行转换为无序列表。

## 安装
```shell
npm install @tiptap/extension-bullet-list @tiptap/extension-list-item
```


## 用法示例
<Fieldset>
  <BulletListDemo />
</Fieldset>

<CollapseCodeblock language="jsx">{BulletListDemoJs}</CollapseCodeblock>

|方法| 说明|使用方法|
| --- | --- | --- |
| toggleBulletList() | 切换|editor.commands.toggleBulletList()|

|命令| Windows/Linux|macOS|
| --- | --- | --- |
