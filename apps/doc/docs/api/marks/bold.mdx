[Bold](https://github.com/ueberdosis/tiptap/blob/main/packages/extension-bold/)     
使用此扩展以粗体呈现文本。     
如果您在编辑器的初始内容中有`<strong>`标签或带有相关的内联样式的，它们都会相应地呈现。

```shell
npm install @tiptap/extension-bold
```

## API
|方法| 说明|
| --- | --- |
| setBold | 将文本标记为粗体 |
| toggleBold | 切换粗体标记 |
| unsetBold | 删除粗体标记 |

举个例子：
```js
editor.commands.setBold()
```

## 设置
通过configure来对标记Bold进行额外配置
### HTMLAttributes
